Fråga 2

Gör samma sak som förra uppgiften men byt ut filter funktionen mot en loop som filtrerar och skriver ut arrayen. 


const myArray = [1, 4.2, 9, 8.7, 15, 17.6];
let newArray = [];
for (let i = 0; i < myArray.length; i++) {
  if (Number.isInteger(myArray[i])) {
    newArray.push(myArray[i]);
  }
}
console.log(newArray);

Fråga 3

Börja med att skapa en array med dessa nummer 84, 98, 48, 98, 28, 32, 56, 71, 98, 29.
Använd filter för att filtrera bort alla ojämna tal och skapa en funktion som tar emot ett tal och kollar om det är jämt eller inte.
Funktionen returnerar true om numret är jämnt och annars false.
Skriv ut resultatet. 


const myArray = [84, 75, 48, 5, 28,
                32, 56, 71, 98, 29];
console.log(myArray.filter(isEven));

function isEven(number) {
  if (number % 2 === 0) {
    return true;
  } else {
    return false;
  }
}
